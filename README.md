# Misty Step Homebrew Tap

Homebrew formulae for [Misty Step](https://mistystep.io) projects.

## Installation

```bash
brew tap misty-step/tap
```

## Available Casks

### Vibe Machine

Cinematic audio visualization for macOS.

```bash
brew install --cask vibe-machine
```

## Updating

```bash
brew update
brew upgrade --cask vibe-machine
```
